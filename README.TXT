Project: Dreamscape
Authors: Michael Kölling, David J. Barnes, and Rita Costa (k19005287)

This project was written using 'zuul-better'as the starting point.
This 'zuul-better' is part of the material for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2016
   
This project is a simple framework for an adventure game. In this version,
it has a few rooms and the ability for a player to walk between these rooms.

To start this application, call the main method from the class GameMain.
